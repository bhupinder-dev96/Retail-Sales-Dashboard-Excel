# Retail Sales Performance Dashboard (Excel)

## Project Overview

This project analyzes retail sales data using Microsoft Excel.

The dashboard provides insights into:

- Total Sales
- Total Profit
- Total Orders
- Profit Percentage
- Sales by Region
- Monthly Sales Trend
- Product Performance
- Category Performance

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Timeline
- Conditional Formatting
- Excel Tables
- KPI Calculations

## Key Insights

- South region generated the highest sales.
- HP Printer was the top-selling product.
- August recorded the highest monthly sales.
- Overall profit margin was approximately 22%.

## Dashboard Features

- Interactive Region Filter
- Product Filter
- Category Filter
- Timeline Filter
- Dynamic KPI Cards
